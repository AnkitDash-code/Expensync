## Getting Started

### 1. Clone or Download the Project

If you're cloning:

```bash
git clone https://github.com/THAT-IT-TEAM/Mobile_Frontend.git
cd Mobile_Frontend
```
---

2. Setup Flutter Environment
Make sure Flutter is installed. Check with:

```bash
flutter --version
```

Install dependencies:

```bash
flutter pub get
```

3. Run the App
   ```bash
   flutter run
   ```
